輪廓擬合

常用幾何圖形擬合技術包括 :

    1.矩形擬合：以矩形逼近contour

    2.曲線擬合：以圓或橢圓逼近contour
-----------------------------------------------------------------------------------------------------------------------------------------------------

矩形擬合 :

    🔹 包圍矩形(Bounding Rectangle)

	retval = cv2.boundingRect(array)

 		🔸 array: input contour

 		🔸 retval: (x,y,w,h),

 		🔸 (x,y): top-left corner coordinate of the BB

 		🔸 w:widthoftheBB

 		🔸 h:height of the BB

    🔹 最小包圍矩形(Minimal Bounding Rectangle)
	
	retval = cv2.minAreaRect(points)

  		🔸 points: a set of 2D points (contour)

  	retval 為一旋轉矩形(rotated rectangle), 包含的資訊有:

  		🔸 矩形的中心座標

  		🔸 矩形的長與寬

  		🔸 矩形相對於 x 軸的旋轉角度（以度數為單位）
-----------------------------------------------------------------------------------------------------------------------------------------------------

曲線擬合 :

    🔹 最小包圍圓形

	center, radius = cv2.minEnclosingCircle(points)

 		🔸 Points: data coordinate set like contour

 		🔸 center: 圓中心座標(x,y)

 		🔸 radius: 代表圓的半徑

    🔹 最優擬合橢圓

	retval = cv2.fitEllipse(points)

 		🔸 points: input set of 2D points (contour)

	 retval：一個元組（tuple），包含了最佳擬合橢圓（best-fit ellipse）的參數。這些參數包括：

		🔸橢圓中心的座標 (x, y)

		🔸橢圓的長軸與短軸長度 (a, b)

		🔸橢圓的旋轉角度 θ（以度為單位，也就是長軸與 x 軸之間的夾角）
